# AI Code Uploader

智能文件引用与 AI 对话助手 - VS Code 插件
当大家都在用cli的时候，这里有一个古朴的对话编程解决方案：适合订阅了对话版而非api版本的省钱玩家。
现在支持批量通过文件名导入

## 功能特性

- **📝 代码选中引用**：选中代码后右键 → "发送到AI"，或使用快捷键 `Ctrl+Shift+A`
- **📁 文件/目录引用**：添加文件或整个目录到 AI 指令
- **📋 粘贴路径添加**：直接粘贴文件路径（绝对路径或相对路径）快速添加文件
- **🔗 智能行号合并**：同一文件不同行号自动合并，避免重复
- **📤 一键复制指令**：生成标准格式指令并复制到剪贴板
- **🌐 快速打开网页**：支持 Kimi、ChatGPT、Claude、Gemini、通义千问等主流 AI 平台
- **💾 导出文件**：将指令保存为 Markdown 文件，或批量导出引用的文件
- **⚡ 快捷用语**：自定义常用指令，支持分类管理和批量插入
- **🔄 覆盖文件**：从导出的文件列表按原目录结构覆盖回原路径

## 使用方法

### 1. 添加代码引用

**方式一：选中代码**
- 在编辑器中选中代码，右键选择 **"发送到AI"**
- 或使用快捷键 `Ctrl+Shift+A` (Mac: `Cmd+Shift+A`)

**方式二：粘贴文件路径**
- 在 AI Uploader 面板的输入框中粘贴文件路径
- 支持绝对路径（如 `G:\project\src\file.ts`）
- 支持相对路径（如 `src/panels/AIPanelProvider.ts`）
- 支持逗号分隔的多个路径
- 按回车或点击 + 按钮添加

**方式三：拖拽文件**
- 直接从文件资源管理器拖拽文件到 AI Uploader 面板

**方式四：从工作区选择**
- 点击"从工作区选择"按钮，选择要添加的文件

### 2. 管理引用

- 在侧边栏 AI Uploader 面板中查看所有引用
- 点击文件旁的 ✕ 移除单个引用
- 点击"清空"按钮移除所有引用

### 3. 使用快捷用语

- 点击"快捷用语"按钮打开管理面板
- 支持分类管理（如默认分类、代码审查、重构等）
- 点击 + 按钮将单条用语插入到指令区域
- 多选模式下可批量添加多条用语

### 4. 配置 AI 平台

- 点击"管理"按钮打开站点管理
- 支持新增、编辑、删除 AI 平台
- 常用预设：Kimi、ChatGPT、Claude、Gemini、通义千问

### 5. 使用指令

| 功能 | 说明 |
|------|------|
| **复制指令** | 点击"复制指令"按钮，粘贴到 AI 对话平台 |
| **打开网页** | 点击"打开网页"选择 AI 平台，自动打开并复制指令 |
| **导出MD** | 点击"导出MD"将指令保存为 .md 文件 |
| **批量导出** | 将所有引用的文件导出到指定目录 |
| **覆盖文件** | 从导出的文件按原目录结构覆盖回原路径 |

## 指令格式示例

```markdown
## 目录结构

├── src/
│   ├── panels/
│   │   └── AIPanelProvider.ts
│   └── extension.ts

## 引用文件

- AIPanelProvider.ts (1-100)
- extension.ts (1-50)

## 需求描述

请分析以上文件并提供帮助。
```

## 安装

### 方式一：从 VSIX 安装

1. 下载 `.vsix` 文件
2. 在 VS Code 中按 `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)
3. 输入并选择 `Extensions: Install from VSIX`
4. 选择下载的 `.vsix` 文件

### 方式二：从市场安装（待发布）

在 VS Code 扩展市场中搜索 "AI Code Uploader"
